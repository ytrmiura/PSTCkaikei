１．ファイルの説明
	fview.c : ファイルのサイズとファイルの種類と最終編子時刻を表示するプログラム
	catfiles.c : 複数のテキストファイルの内容を結合して、一つのファイルにまとめるプログラム

2.コンパイルの方法
	fview.c ⇒　gcc fview.c -o fview
	catfiles.c ⇒ gcc catfiles.c -o catfiles

3.実行方法
	fview.c : ./fview aaa.txt bbb.txt ccc.txt ・・・
	catfiles.c : ./catfiles aaa.txt bbb.txt ccc.txt ・・・
